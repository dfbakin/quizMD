### Размещения и сочетания (Базовый уровень)

В каком из объектов порядок выбранных элементов из $n$ по $k$ учитывается?

- [ ] В сочетаниях $C_n^k$
- [x] В размещениях $A_n^k$
- [ ] В количестве подмножеств $2^n$
- [ ] В числе Каталана

> В размещениях порядок важен, а в сочетаниях — нет.

### Вычисление размещений (Базовый уровень)

Чему равно $A_8^3$?

- [ ] $56$
- [x] $336$
- [ ] $512$
- [ ] $40320$

> $A_8^3 = 8 \cdot 7 \cdot 6 = 336$ (эквивалентно $\frac{8!}{(8 - 3)!}$).

### Вычисление сочетаний (Базовый уровень)

Чему равно $C_{10}^3$?

- [ ] $720$
- [x] $120$
- [ ] $90$
- [ ] $45$

> $C_{10}^3 = \frac{10!}{3! \cdot 7!} = 120$.

### Связь размещений и сочетаний (Средний уровень)

Какие равенства верны? (выберите все верные)

- [x] $A_n^k = C_n^k \cdot k!$
- [ ] $C_n^k = \frac{A_n^k}{(n - k)!}$
- [x] $C_n^k = \frac{A_n^k}{k!}$
- [ ] $A_n^k = C_n^k + k!$

> Из формулы $A_n^k = C_n^k \cdot k!$ сразу следует $C_n^k = \frac{A_n^k}{k!}$. Варианты 2 и 4 неверны.

### Генерация перестановок рекурсией (Средний уровень)

В алгоритме генерации перестановок через обмены $a[pos] \leftrightarrow a[j]$ что нужно сделать после возврата из рекурсивного вызова?

```cpp
void permute(vector<int>& a, int pos) {
    if (pos == a.size()) { print(a); return; }
    for (int j = pos; j < a.size(); j++) {
        swap(a[pos], a[j]);
        permute(a, pos + 1);
        // что здесь?
    }
}
```

- [ ] Отсортировать суффикс
- [x] Выполнить обратный обмен тех же элементов
- [ ] Удалить последний элемент
- [ ] Ничего не делать

> Нужно откатить состояние (backtracking), чтобы корректно перейти к следующей ветке перебора.
